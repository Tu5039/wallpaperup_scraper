# Wallpaperup 爬虫

## 环境
python version: python3.7.5
core dependency: bs4 requests lxml
other dependency: traceback os sys
editor: vscode

## 构建环境
```shell
pip install bs4 requests lxml
```
## 运行
```shell
python wallpaperup_scrapy.py
```
